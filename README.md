# NCurses Sorting Visualizer

---

## Build
You can build the program by running `make`

## Options
   - a              Enable all sorts.
   - b              Enable Bubble Sort.
   - s              Enable Shell Sort.
   - q              Enable Quick Sort.
   - h              Enable Heap Sort.
   - r seed         Specify random seed.

> EX: `./sorting -h -r 1337` will run heap sort with a seed of 1337.
